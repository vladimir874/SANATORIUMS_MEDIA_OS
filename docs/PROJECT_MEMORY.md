# Память проекта и принятые решения

Этот файл сводит решения из разрозненных чатов, исходного migration pack,
аудита Olymp IV и текущего репозитория. Он является навигацией по версиям, но
не заменяет `project_status.json` как оперативный статус.

## Линия версий

1. Исходный `hotelcut-chatgpt-memory.md` описал четырехэтапный замысел:
   ExifTool/FFmpeg, сцены/QC, XML-таймлайн и аудит.
2. Аудит SANATORIUMS MEDIA FACTORY уточнил реальную структуру и медиаматериалы.
3. Архитектурная проверка Premiere разделила локальный анализ и тонкий UXP-слой;
   FCPXML перестал считаться гарантированным прямым импортом.
4. HOTELCUT 0.1.0 реализовал и проверил шаги 1.1 и 1.2.
5. Версия 0.2.0 объединяет рабочее ядро, Media OS и историю решений как
   финальную тестовую демо-версию текущего утвержденного объема.

Оригинальный migration pack сохранен в `docs/history/` как архивный источник.
При конфликте приоритет имеют проверенный код, реальные отчеты и этот документ.

## Подтвержденные данные Olymp IV

- 134 MP4, все исходники доступны и не изменились на момент проверки.
- 127 Canon: H.264, 1920×1080, 50 fps, с аудиодорожкой.
- 7 DJI: HEVC, 3840×2160, 60000/1001 fps, без аудиодорожки.
- Общая длительность: 1850.925251 секунды (00:30:50.925).
- Метаданные прочитаны ExifTool 13.59: 134 успешно, 0 ошибок.
- У Canon присутствует tmcd-трек, но стартовый timecode не читается; это warning.
- У DJI временная зона не установлена; она не подставляется предположением.

## Утверждено пользователем

- Шаг 1.1: безопасное сканирование папок и manifest.
- Шаг 1.2: единичный запуск ExifTool и нормализованный metadata manifest.
- Следующий шаг после приемки демо: 1.3, локальные FFmpeg-прокси.

## Контекст Premiere и графики

- Целевая монтажная среда — Premiere Pro 2026.
- Предыдущий шаблон Premiere 15.4.1 содержит 1080p50-последовательность,
  интро/музыку и старые пути `E:\`; его нельзя выдавать за готовый шаблон 2026.
- Голосовая начитка для voiceover-driven монтажа в проверенном наборе отсутствует.
- В графическом архиве найдены редактируемые PSD-мастера обычного титра, номера,
  должности, URL и footer-вариантов; часть `title-sanatoriums_NEW_*` растровая.
- Старые JSX в `experiments/after-effects/` — прототипы After Effects, не UXP.

## Творческие ориентиры

Сверялись Humboldt Park Hotel & Spa (09:35), Vila Livia (11:48) и Schlosspark
(08:22). Из них подтвержден только общий принцип секционного рассказа,
постоянной музыкальной/графической системы и покрытия речи B-roll. Автоматический
творческий порядок на шагах 1.1–1.2 намеренно не формируется.

## Открытые ограничения

- FFmpeg в проверенной среде не найден.
- Этапы 2–4 не реализованы.
- Реальный импорт/сборка UXP не тестировались в Premiere.
- Для монтажа по начитке нужны финальный WAV и разбивка текста по смысловым блокам.
