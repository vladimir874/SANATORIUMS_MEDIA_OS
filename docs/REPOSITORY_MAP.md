# Карта репозитория

| Путь | Назначение | Статус |
|---|---|---|
| `src/hotelcut/` | сканирование и метаданные | production для шагов 1.1–1.2 |
| `tests/` | unit, integration и demo-contract | обязательная проверка |
| `config/` | редактируемая карта папок | source of truth |
| `src/hotelcut/data/` | копия карты внутри Python-пакета | должна совпадать с `config/` |
| `vendor/exiftool/` | переносимый ExifTool 13.59 | Windows runtime |
| `vendor/ffmpeg/` | локальный FFmpeg и сведения о сборке | exe игнорируются Git |
| `outputs/` | обезличенные эталонные отчеты | можно публиковать |
| `private_outputs/` | локальные полные отчеты | игнорируется Git |
| `00_Core/` | стандарты SANATORIUMS MEDIA OS | база знаний |
| `01_Agents/` | роли и регламенты помощников | база знаний |
| `apps-script/` | трекер Google Sheets | отдельный вспомогательный модуль |
| `prompts/`, `templates/`, `examples/` | материалы трекера | вспомогательные |
| `experiments/after-effects/` | старые JSX-прототипы | не часть HOTELCUT runtime |
| `docs/history/` | исходные архивные документы | не нормативная спецификация |
| `work/` | временные/приватные файлы | игнорируется Git |

В корне оставлены только входные документы проекта, конфигурация сборки, статус
и навигация. Это уменьшает риск перепутать прототип JSX с рабочим Premiere UXP.
