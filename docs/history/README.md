# Архив исходных решений

`hotelcut-chatgpt-memory-original.md` — исходный migration pack из прошлых
переписок. Он сохранен без смысловой переработки для трассируемости.

Документ не является нормативной спецификацией. В частности, его XML-пример
смешивает FCPXML и legacy XMEML, а некоторые библиотеки были предложены до
реальной проверки. Актуальные решения находятся в `docs/ARCHITECTURE.md`,
`docs/PROJECT_MEMORY.md` и `project_status.json`.
